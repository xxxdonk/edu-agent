# EduAgent 比赛演示脚本

目标时长：6 分 30 秒，最大不超过 7 分钟。

演示课程：《机器学习基础》。
演示前确认后端、前端、模型服务和数据库健康检查全部通过。

## 0:00-0:30 开场与问题

画面：EduAgent 学习工作台首屏。

讲解词：

“EduAgent 是一个由动态学习画像驱动的多智能体个性化学习资源生成系统。它不依赖传统问卷，而是从自然语言学习对话开始，完成画像、路径、五类资源、评价和动态调整的完整闭环。”

操作：指向顶部三段流程导航和绿色后端健康状态。

## 0:30-1:35 自然语言画像

操作：输入以下演示文本并发送：

> 我是计算机科学专业大二学生，正在学习机器学习，了解 Python，但梯度下降比较薄弱。我偏好中文讲解、思维导图和代码实践，每天可以学习 45 分钟、每周 5 天，目标是完成课程项目。

讲解词：

“Profile Agent 从对话中抽取事实和推断，不仅给出画像值，还为每个字段保存证据来源和置信度。页面现在展示专业、课程、知识水平、目标、薄弱点、学习历史、认知风格、语言偏好、资源偏好和时间预算共 10 个维度。”

操作：点击“薄弱知识点”的“查看字段证据”，展示原始对话引用；指出画像版本和模式标签。

## 1:35-2:20 个性化学习路径

操作：进入“个性化学习路径工作区”。

讲解词：

“Planner Agent 根据画像版本、薄弱点、目标和每天可用时间生成有先后关系的学习路径。每一步都有学习目标、安排原因、推荐资源、前置知识、预计时间和完成标准，而不是简单的资源列表。”

操作：展开浏览两个步骤，选择第一步作为当前学习目标。

## 2:20-3:40 多智能体生成

操作：进入“资源中心与学习评价”，点击“生成五类资源”。

讲解词：

“Orchestrator 通过共享画像和路径上下文并行调度五个资源 Agent。页面通过 SSE 实时接收可持久化事件，展示检索、讲解、思维导图、练习、阅读、代码和 Reviewer 审校状态。单个 Agent 失败不会让整个任务崩溃，任务会进入部分成功并保留已有资源。”

操作：跟随生成轨迹指出 `waiting/running/completed/failed`；任务完成后指出接收事件数量。

## 3:40-4:50 五类资源与来源审校

操作：依次点击五个资源标签。

讲解词：

“这里是 Markdown 课程讲解、Mermaid 思维导图、分层练习、拓展阅读和 Python 代码实践。每项资源都带有目标知识点、难度、个性化原因、知识库来源、审校状态和创建时间。”

操作：

1. 展示课程讲解的 Markdown 结构。
2. 展示 Mermaid 思维导图。
3. 展开知识库来源与定位。
4. 展示 Python 代码高亮。
5. 指出 Reviewer 的“审校通过”状态。

## 4:50-5:55 练习评价

操作：在分层练习中故意答错一题，其余正常回答，点击提交评价。

讲解词：

“Evaluation Agent 返回掌握度、是否通过、薄弱知识点和反馈。前端同时展示每题结果、参考答案和解析，并保留学生答案。”

操作：展示总分、逐题反馈、薄弱知识点和学习建议。

## 5:55-6:25 动态更新闭环

讲解词：

“当评价要求调整时，评价证据会进入新的画像版本，Planner 再结合旧路径生成调整后的路径。这里可以直接比较更新前后的画像字段和学习步骤，形成可解释、可追踪的个性化闭环。”

操作：展示画像更新前后与路径更新前后对比。

## 6:25-6:30 收尾

讲解词：

“EduAgent 的核心价值，是让个性化学习从一次推荐升级为由证据和评价持续驱动的多智能体学习过程。”

## 演示前检查清单

- 后端 `/api/health` 返回 `status=ok`、`database=ok`
- 前端右上角为绿色健康状态
- `Agent2` 资源与评价代码已经合入演示分支
- 真实 LLM 配置成功，页面显示“结构化大模型结果”
- 预先跑通一次新 student_id，确认五资源与 Mermaid 可渲染
- 浏览器缩放 100%，窗口至少 1366×768
- 关闭通知、即时通信和自动更新弹窗
- 准备一份已跑通的数据库作为断网恢复预案，但现场不使用固定前端结果

## 故障应对

- 真实模型超时：说明系统进入公开标记的开发适配器，继续展示稳定闭环，不声称为真实模型结果。
- 单个资源失败：展示 `partial_success` 和错误项，再打开已成功资源，突出失败隔离。
- SSE 中断：页面会按 `after=sequence` 重连，并使用任务轮询兜底。
- Mermaid 失败：展示中文错误和原始 Mermaid 内容，其他资源不受影响。
