# 支持向量机

## 学习目标
1. 理解最大间隔分类器的几何直觉与数学推导
2. 掌握线性可分 SVM 的硬间隔优化问题及其对偶形式
3. 理解软间隔 SVM 如何通过松弛变量处理线性不可分数据
4. 掌握核技巧的基本思想及常见核函数（线性核、RBF 核、多项式核）
5. 能够分析 SVM 的优缺点并在实际问题中合理选用

## 核心概念
- **支持向量机 (SVM)**：通过寻找最大化分类间隔的超平面来实现分类的算法。核心思想是在特征空间中构造最优分离超平面，使不同类别的样本到超平面的最小距离最大化。
- **最大间隔 (Maximum Margin)**：分类超平面到最近样本点的距离。大间隔意味着分类器对数据扰动和噪声有更强的鲁棒性。
- **支持向量 (Support Vectors)**：距离最优超平面最近的那些训练样本点。SVM 的解仅由支持向量决定，其他样本对模型无影响——这一稀疏性质是 SVM 的重要特征。
- **软间隔 (Soft Margin)**：通过引入松弛变量 $\xi_i \geq 0$，允许部分样本被错分或落在间隔内部，在最大化间隔和最小化分类错误之间折中。正则化参数 C 控制这一折中程度。
- **核函数 (Kernel Function)**：$K(\mathbf{x}_i, \mathbf{x}_j) = \langle \phi(\mathbf{x}_i), \phi(\mathbf{x}_j) \rangle$，隐式地在高维空间计算内积而无需显式映射，使得 SVM 能高效处理非线性可分问题。
- **对偶问题 (Dual Problem)**：通过拉格朗日对偶性将原始优化问题转换为其对偶形式，使得优化仅依赖于样本间的内积——这是应用核技巧的关键前提。
- **RBF 核 (高斯核)**：$K(\mathbf{x}_i, \mathbf{x}_j) = \exp(-\gamma ||\mathbf{x}_i - \mathbf{x}_j||^2)$，最常用的核函数，对应无限维特征空间。$\gamma$ 参数控制高斯函数的宽度，影响决策边界的平滑度。
- **KKT 条件**：Karush-Kuhn-Tucker 条件，是约束优化问题最优解的必要条件。在 SVM 的对偶推导中起关键作用。

## 知识正文
支持向量机是 20 世纪 90 年代统计学习理论的巅峰成果，由 Vapnik 等人系统发展。其核心思路简洁而深刻：在众多能将两类样本分开的超平面中，选择具有最大几何间隔的那个。直觉上，间隔越大，分类器对新数据的泛化能力越强——这得到了 VC 维理论的严格支撑。

对于线性可分情况，SVM 的硬间隔优化问题可表述为：最小化 $\frac{1}{2}||\mathbf{w}||^2$，满足约束 $y_i(\mathbf{w}^T\mathbf{x}_i + b) \geq 1$。目标函数最小化权重的 L2 范数等效于最大化几何间隔 $2/||\mathbf{w}||$。这是一个凸二次规划问题，有全局唯一最优解（假设数据线性可分）。

通过拉格朗日乘子法将原始问题转化为对偶问题后，优化变量从 d 维（特征数）变为 n 维（样本数）。对偶形式为最大化 $\sum_i \alpha_i - \frac{1}{2}\sum_{i,j}\alpha_i\alpha_j y_i y_j \mathbf{x}_i^T\mathbf{x}_j$，约束 $\alpha_i \geq 0$ 且 $\sum_i \alpha_i y_i = 0$。最终决策函数 $f(\mathbf{x}) = \text{sign}(\sum_{i} \alpha_i y_i \mathbf{x}_i^T \mathbf{x} + b)$，其中非零的 $\alpha_i$ 对应支持向量。

现实数据很少完美线性可分，软间隔 SVM 通过松弛变量 $\xi_i$ 允许一定程度的误分类：约束放松为 $y_i(\mathbf{w}^T\mathbf{x}_i + b) \geq 1 - \xi_i$，目标函数增加惩罚项 $C\sum_i \xi_i$。超参数 C 在间隔最大化和训练误差最小化之间平衡：C 越大，对误分类惩罚越重，模型越倾向拟合训练数据（可能过拟合）；C 越小，模型越倾向简单（可能欠拟合）。

核技巧是 SVM 处理非线性问题的关键创新。对于线性不可分数据，将其映射到高维空间后可能变得线性可分。但显式映射的计算代价可能极高甚至不可能（如 RBF 核对应无限维）。注意到对偶问题中样本仅以内积形式出现：$\mathbf{x}_i^T\mathbf{x}_j$，如果存在函数 K 使得 $K(\mathbf{x}_i, \mathbf{x}_j) = \phi(\mathbf{x}_i)^T \phi(\mathbf{x}_j)$，则无需显式计算 $\phi$。Mercer 定理给出了核函数有效性的充要条件。

常见核函数包括：线性核 $K(\mathbf{x}_i, \mathbf{x}_j) = \mathbf{x}_i^T\mathbf{x}_j$（等价于无映射）；多项式核 $K(\mathbf{x}_i, \mathbf{x}_j) = (\gamma \mathbf{x}_i^T\mathbf{x}_j + r)^d$；RBF 核 $K(\mathbf{x}_i, \mathbf{x}_j) = \exp(-\gamma ||\mathbf{x}_i - \mathbf{x}_j||^2)$。RBF 核最通用，仅有两个关键超参数（C 和 $\gamma$），在缺乏先验知识时是很好的默认选择。

SVM 的主要优点：有严谨的数学理论支撑（统计学习理论的 VC 维与结构风险最小化）、解具有稀疏性（仅依赖支持向量）、核技巧赋予强大的非线性处理能力、在高维数据和小样本场景表现优秀。主要缺点：大规模数据下训练效率低（对偶问题涉及 n×n 的核矩阵）、对核函数和超参数敏感需要仔细调参、概率输出非原生需额外校准（Platt Scaling）、对缺失数据敏感。

## 关键公式
- **硬间隔原问题**：$$\min_{\mathbf{w}, b} \frac{1}{2}||\mathbf{w}||^2 \quad \text{s.t.} \quad y_i(\mathbf{w}^T\mathbf{x}_i + b) \geq 1, \forall i$$。
- **软间隔原问题**：$$\min_{\mathbf{w}, b, \xi} \frac{1}{2}||\mathbf{w}||^2 + C\sum_{i=1}^{n}\xi_i \quad \text{s.t.} \quad y_i(\mathbf{w}^T\mathbf{x}_i + b) \geq 1 - \xi_i, \xi_i \geq 0$$，C 为正则化参数。
- **对偶问题**：$$\max_{\alpha} \sum_{i=1}^{n}\alpha_i - \frac{1}{2}\sum_{i,j}\alpha_i\alpha_j y_i y_j K(\mathbf{x}_i, \mathbf{x}_j) \quad \text{s.t.} \quad 0 \leq \alpha_i \leq C, \sum_i \alpha_i y_i = 0$$。
- **RBF 核**：$$K(\mathbf{x}_i, \mathbf{x}_j) = \exp\left(-\frac{||\mathbf{x}_i - \mathbf{x}_j||^2}{2\sigma^2}\right) = \exp(-\gamma ||\mathbf{x}_i - \mathbf{x}_j||^2)$$，$\gamma = 1/(2\sigma^2)$。
- **决策函数**：$$f(\mathbf{x}) = \text{sign}\left(\sum_{i=1}^{n} \alpha_i y_i K(\mathbf{x}_i, \mathbf{x}) + b\right)$$，仅支持向量对应的 $\alpha_i > 0$。

## 示例
**示例 1：线性可分 SVM**  
两类二维点：正类 (1,1), (2,2), (2,0)；负类 (0,0), (1,0), (0,1)。硬间隔 SVM 找到的最优超平面为 x = y，支持向量为 (1,1) 和 (1,0)，间隔为 $\sqrt{2}$。任何平行于该超平面且间隔更大的分离面都不存在。

**示例 2：RBF 核处理同心圆数据**  
内圈半径 1 为正类，外圈半径 3 为负类（环宽 2 的圆环）。线性 SVM 无法分离（不存在一条直线能分开内外圈）。使用 RBF 核（$\gamma=1$），SVM 将数据映射到高维空间后找到分离超平面。决策边界在原始空间中呈现为近似圆形。减小 $\gamma$ 会使边界更平滑（偏向欠拟合），增大 $\gamma$ 会使边界更曲折（容易过拟合）。

## 常见误区
1. **"SVM 只能做二分类"**：错误。通过 OvR 或 OvO（一对一投票）策略，SVM 可以处理多分类问题。OvO 训练 K(K-1)/2 个分类器，在 Scikit-learn 的 SVC 中默认使用。
2. **"核函数越多越好"**：错误。过多的核函数参数或不当的核选择会导致过拟合或计算开销过大。RBF 核通常是最安全的第一选择，仅在特征数远大于样本数时优先考虑线性核。
3. **"大 C 值总是更好"**：错误。C 越大对误分类惩罚越重，决策边界越贴合训练数据，容易过拟合噪声。过大的 C 还会使优化问题病态。
4. **"SVM 不需要特征缩放"**：错误。SVM 对特征尺度非常敏感（基于距离/内积），不同量纲的特征会扭曲核函数计算结果。使用 SVM 前必须标准化或归一化特征。
5. **"支持向量越多模型越差"**：不准确。支持向量数量反映问题的内在难度而非模型质量。线性可分数据只需少量支持向量，高度重叠的复杂数据自然需要更多。

## 基础练习
**题目 1（单选）**：SVM 中正则化参数 C 的作用是：
A. 控制核函数的宽度
B. 控制间隔最大化和训练误差最小化之间的权衡
C. 控制多项式核的阶数
D. 控制迭代次数
**正确答案**：B
**解析**：C 是软间隔 SVM 的惩罚系数。C 大 → 对误分类惩罚重 → 更关注减少训练误差（可能过拟合）；C 小 → 更关注最大化间隔（可能欠拟合）。$\gamma$ 控制 RBF 核的宽度，d 控制多项式核的阶数。

**题目 2（单选）**：使用 RBF 核 SVM 时训练准确率 100% 但测试准确率很低，以下哪个调整最可能改善泛化性能？
A. 增大 C
B. 减小 C
C. 增大 $\gamma$
D. 使用更高阶多项式核
**正确答案**：B
**解析**：训练 100% 但测试很差是过拟合的典型表现。减小 C 降低对训练误差的惩罚，让模型更关注最大化间隔，从而提升泛化能力。增大 C 或 $\gamma$ 都会加剧过拟合。

## 进阶练习
**题目**：从对偶问题的角度解释为什么核技巧能够使 SVM 高效处理非线性问题。并说明 RBF 核参数 $\gamma$ 对决策边界的影响及如何选择合适的 $\gamma$。

**参考答案**：
对偶问题的优化目标仅依赖样本间的内积 $\mathbf{x}_i^T\mathbf{x}_j$，而非样本的显式坐标。核技巧利用这一点，用核函数 $K(\mathbf{x}_i, \mathbf{x}_j)$ 替换内积，隐式实现了 $\phi(\mathbf{x}_i)^T\phi(\mathbf{x}_j)$ 的计算。这意味着 SVM 在原始空间中做出非线性决策边界，但在隐式映射的高维特征空间中实际构建的是一个线性分类器。整个过程无需显式计算或存储 $\phi(\mathbf{x})$，将计算复杂度保持在样本数级别而非特征空间维度。

$\gamma$ 对 RBF 核的影响：$\gamma$ 越大，$K$ 值衰减越快，每个样本的"影响范围"越小，决策边界越复杂扭曲，容易过拟合；$\gamma$ 越小，样本影响范围越大，决策边界越平滑，可能欠拟合。可类比于 KNN 中 k 的作用：小 $\gamma$ 相当于大 k（考虑更多邻居），大 $\gamma$ 相当于小 k（只考虑最近的邻居）。

选择策略：在 log 尺度上对 C 和 $\gamma$ 做网格搜索（如 $[10^{-3}, 10^3]$），结合交叉验证选择最优组合。典型范围 C ∈ [0.1, 100]，γ ∈ [0.001, 1]。也可利用启发式：$\gamma = 1/d$（d 为特征数）作为起点。
