# 机器学习基础 — 课程大纲

## 1. 机器学习概述
- 机器学习的定义与分类（监督学习、无监督学习、强化学习）
- 机器学习的基本流程：数据 → 模型 → 评估 → 部署
- 过拟合与欠拟合的概念
- 偏差-方差权衡

## 2. 线性回归
- 模型假设：y = wx + b
- 损失函数：均方误差（MSE）
- 解析解：正规方程
- 梯度下降求解

## 3. 逻辑回归与分类
- Sigmoid 函数与概率输出
- 交叉熵损失函数
- 二分类与多分类
- 决策边界可视化

## 4. 支持向量机（SVM）
- 最大间隔分类器
- 核函数：线性核、RBF 核
- 软间隔与正则化
- SVM 的优缺点

## 5. 决策树与集成学习
- ID3 / C4.5 / CART 算法
- 信息增益与基尼系数
- 随机森林
- GBDT / XGBoost / LightGBM

## 6. 神经网络基础
- 感知机模型
- 多层感知机（MLP）
- 激活函数：ReLU、Sigmoid、Tanh
- 反向传播算法

## 7. 卷积神经网络（CNN）
- 卷积层、池化层、全连接层
- LeNet-5、AlexNet、VGG、ResNet
- 迁移学习
- 图像分类实战

## 8. 循环神经网络（RNN）
- RNN 基本结构
- LSTM 与 GRU
- 序列到序列模型
- 文本生成与机器翻译

## 9. 无监督学习
- K-Means 聚类
- 层次聚类
- PCA 降维
- t-SNE 可视化

## 10. 模型评估与调优
- 交叉验证
- 混淆矩阵、精确率、召回率、F1
- ROC 曲线与 AUC
- 超参数调优（网格搜索、随机搜索、贝叶斯优化）
