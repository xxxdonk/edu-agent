# 线性回归

## 学习目标
1. 理解线性回归的模型假设与参数含义
2. 掌握最小二乘法的推导逻辑和正规方程解析解
3. 理解梯度下降优化方法的原理及与正规方程的区别
4. 能够评估线性回归模型的性能并诊断常见问题
5. 了解多元线性回归、多项式回归与正则化变体（岭回归、Lasso）

## 核心概念
- **线性回归 (Linear Regression)**：假设目标变量 Y 与特征 X 之间存在线性关系 $Y = wX + b + \epsilon$，通过学习参数 w（权重）和 b（偏置）来最小化预测误差。
- **均方误差损失 (MSE Loss)**：$L = \frac{1}{n}\sum_{i=1}^{n}(y_i - \hat{y}_i)^2$，是线性回归最常用的损失函数，对异常值较为敏感。
- **正规方程 (Normal Equation)**：$\mathbf{w} = (X^T X)^{-1} X^T \mathbf{y}$，通过矩阵运算直接求得最优参数的解析解，无需迭代。但当特征数量很大时矩阵求逆计算开销大。
- **梯度下降 (Gradient Descent)**：通过迭代沿负梯度方向更新参数来逐步逼近最优解。参数更新公式 $w := w - \alpha \frac{\partial L}{\partial w}$，其中 $\alpha$ 为学习率。
- **多元线性回归**：将一元线性回归推广到多个特征，模型为 $\hat{y} = w_0 + w_1x_1 + w_2x_2 + \dots + w_dx_d$，可用矩阵形式统一表示为 $\mathbf{\hat{y}} = X\mathbf{w}$。
- **R² 决定系数**：$R^2 = 1 - \frac{\sum(y_i - \hat{y}_i)^2}{\sum(y_i - \bar{y})^2}$，衡量模型解释的方差比例，越接近 1 表示拟合越好。
- **岭回归 (Ridge)**：在线性回归损失函数中加入 L2 正则项 $\lambda\sum w_j^2$，缓解共线性和过拟合，使参数收缩但不为零。
- **Lasso 回归**：加入 L1 正则项 $\lambda\sum |w_j|$，可产生稀疏解，即自动将不重要特征的参数压缩为零，兼具特征选择功能。

## 知识正文
线性回归是最基础也是最经典的监督学习算法，几乎所有机器学习教程都以此为起点。它假定目标变量与输入特征之间存在线性关系，数学表达式为 $\hat{y} = w_1x_1 + w_2x_2 + \dots + w_dx_d + b$，向量形式为 $\hat{y} = \mathbf{w}^T\mathbf{x} + b$。学习的任务就是找到最优的 w 和 b，使预测值与真实值的误差最小。

最小二乘法是线性回归参数估计的经典方法。其思想是选择参数使得所有训练样本的残差平方和最小。将损失函数 $L(\mathbf{w}) = \frac{1}{2}\sum_{i=1}^{n}(y_i - \mathbf{w}^T\mathbf{x}_i)^2$ 对 w 求导并令导数为零，可得到正规方程。对于矩阵 X（每一行为一个样本，每一列为一个特征）和标签向量 y，正规方程的解为 $\mathbf{w}^* = (X^T X)^{-1} X^T \mathbf{y}$。

正规方程的优势在于一步到位，无需选择学习率或迭代。但它有两个主要局限：一是当 $X^T X$ 不可逆（如特征之间存在完全共线性或特征数大于样本数）时无法直接使用；二是矩阵求逆的时间复杂度为 $O(d^3)$（d 为特征数），当 d 很大时计算代价极高，这在高维数据场景下不可接受。

梯度下降通过迭代优化的方式解决上述问题。参数更新规则为每次将所有参数沿着损失函数梯度的反方向移动一小步。学习率 α 是关键超参数：过大可能导致震荡甚至发散，过小则收敛缓慢。实践中常使用小批量梯度下降（Mini-batch GD）来平衡计算效率和收敛稳定性，以及在特征尺度不一致时先做标准化以加速收敛。

多元线性回归将一元形式自然扩展到多维特征空间。此时 w 不再是一个标量而是一个向量，决策边界是一个超平面。需要注意的实践要点包括：特征尺度标准化（避免不同量纲特征对梯度的影响不均）、处理分类特征（独热编码）、检测并处理多重共线性（通过方差膨胀因子 VIF 诊断）。

评估线性回归模型时，MSE 和 MAE（平均绝对误差）提供绝对误差度量，R² 提供相对度量。但需注意，特征数增加时 R² 必然不降，因此引入调整 R² 来惩罚不必要的特征。残差图（Residual Plot）是诊断模型是否满足线性假设、等方差性和独立性假设的直观工具。

当数据存在共线性或需要防止过拟合时，正则化线性回归是自然的升级路径。岭回归在损失函数中引入 L2 惩罚项，等价于对参数施加高斯先验；Lasso 引入 L1 惩罚项，等价于拉普拉斯先验，能产生稀疏解。弹性网络（Elastic Net）结合两者，兼具变量选择和参数收缩的优点。

## 关键公式
- **线性模型**：$$\hat{y} = \mathbf{w}^T\mathbf{x} + b = \sum_{j=1}^{d} w_j x_j + b$$，其中 $d$ 为特征维度，$w_j$ 为第 $j$ 个特征的权重，$b$ 为偏置项。
- **正规方程**：$$\mathbf{w} = (X^T X)^{-1} X^T \mathbf{y}$$，其中 $X$ 为 $n \times d$ 设计矩阵（已包含偏置列），$\mathbf{y}$ 为 $n$ 维标签向量。
- **梯度下降更新**：$$w_j^{(t+1)} = w_j^{(t)} - \alpha \frac{\partial L}{\partial w_j}$$，$$\frac{\partial L}{\partial w_j} = \frac{1}{n}\sum_{i=1}^{n} (\hat{y}_i - y_i) x_{ij}$$，$\alpha$ 为学习率。
- **R² 决定系数**：$$R^2 = 1 - \frac{SS_{res}}{SS_{tot}}$$，$SS_{res}$ 为残差平方和，$SS_{tot}$ 为总平方和。
- **岭回归损失**：$$L_{ridge} = \frac{1}{n}\sum_{i=1}^{n}(y_i - \hat{y}_i)^2 + \lambda \sum_{j=1}^{d} w_j^2$$，$\lambda \geq 0$ 为正则化强度。

## 示例
**示例 1：学生学习时间与成绩**  
设有 5 个学生数据，学习时间（小时）与考试成绩（分）：(1, 55), (2, 62), (3, 71), (4, 78), (5, 88)。使用最小二乘法求解：

计算均值 $\bar{x} = 3$，$\bar{y} = 70.8$。
分子 $\sum (x_i - \bar{x})(y_i - \bar{y}) = (-2)(-15.8) + (-1)(-8.8) + 0 + (1)(7.2) + (2)(17.2) = 31.6 + 8.8 + 7.2 + 34.4 = 82$
分母 $\sum (x_i - \bar{x})^2 = 4 + 1 + 0 + 1 + 4 = 10$
$w = 82/10 = 8.2$，$b = 70.8 - 8.2 \times 3 = 46.2$
回归方程：$\hat{y} = 8.2x + 46.2$，表明每多学习 1 小时，预期成绩提高约 8.2 分。

**示例 2：房价多元线性回归**  
设房价受面积（$x_1$）、卧室数（$x_2$）影响。3 条数据：[(80, 2), 200万], [(120, 3), 350万], [(150, 3), 420万]。通过正规方程（需在 X 中加一列 1 作为偏置项）求解得到参数 w，然后可预测新房屋价格。

## 常见误区
1. **"相关系数为 0 说明没有关系"**：错误。相关系数仅衡量线性关系，两个变量可能存在强烈的非线性关系而线性相关系数接近于零（如 $y = x^2$，x 在 [-1,1] 对称分布时相关系数为 0）。
2. **"R² 高就说明模型好"**：不全面。R² 高可能只是过拟合或多特征堆砌的结果，还需结合调整 R²、残差分析和交叉验证综合判断。
3. **"线性回归要求数据服从正态分布"**：不准确。线性回归假设的是残差（误差项）服从正态分布，而非输入特征或目标变量本身服从正态分布。
4. **"正规方程总是比梯度下降好"**：错误。当特征数 d 很大时（如 d > 10^4），矩阵求逆的 $O(d^3)$ 代价远高于梯度下降的迭代计算。反之 d 较小时正规方程更高效。
5. **"去掉截距项可以让模型更简洁"**：错误。除非有充分的先验知识表明回归线必须过原点，否则不应去掉截距项。强制过原点会使模型产生系统性偏差。

## 基础练习
**题目 1（单选）**：在多元线性回归中，若特征之间存在高度线性相关（多重共线性），使用正规方程会遇到什么问题？
A. 梯度消失
B. $X^T X$ 接近奇异，求逆数值不稳定
C. 学习率需要调大
D. 偏置项会变为零
**正确答案**：B
**解析**：多重共线性导致 $X^T X$ 的列近似线性相关，矩阵条件数很大，接近奇异，求逆时数值不稳定，参数估计方差极大。此时可考虑使用岭回归。

**题目 2（单选）**：梯度下降中学习率设为 0.001，训练过程中损失函数值在相邻迭代间反复剧烈波动，最可能的原因是：
A. 学习率太小
B. 学习率太大
C. 数据未标准化
D. 模型欠拟合
**正确答案**：B
**解析**：损失函数剧烈波动是学习率过大的典型症状——参数在最优解附近反复跳跃甚至发散。学习率太小表现为收敛缓慢但平稳下降。

## 进阶练习
**题目**：解释岭回归 (Ridge) 和 Lasso 回归的核心区别，并说明各适用于什么场景。为什么 Lasso 能产生稀疏解而岭回归不能？

**参考答案**：
岭回归使用 L2 正则化 $\lambda\sum w_j^2$，Lasso 使用 L1 正则化 $\lambda\sum |w_j|$。核心区别在于：L2 正则化在参数空间中对应圆形等高线，最优解通常落在约束边界上但各维参数均不为零；L1 正则化对应菱形约束区域，最优解容易落在坐标轴上，使某些参数恰好为零，因此产生稀疏解。

适用场景：当认为大部分特征都有一定预测作用且不希望任意特征被完全排除时，用岭回归；当怀疑只有少数特征真正重要、希望自动筛选特征时，用 Lasso。在高维数据（特征数远大于样本数）和需要可解释模型时，Lasso 尤其适用。
