# 逻辑回归

## 学习目标
1. 理解逻辑回归的模型形式与概率解释
2. 掌握 Sigmoid 函数的数学性质及其在二分类中的作用
3. 理解交叉熵损失函数的推导逻辑及与 MSE 的区别
4. 掌握从二分类推广到多分类的两种策略：OvR（一对多）和 Softmax 回归
5. 理解决策边界的几何意义及其与模型参数的关系

## 核心概念
- **逻辑回归 (Logistic Regression)**：虽名为"回归"，但实质是分类算法。通过 Sigmoid 函数将线性组合的输出映射到 (0,1) 区间，解释为正类的预测概率。
- **Sigmoid 函数**：$\sigma(z) = \frac{1}{1 + e^{-z}}$，具有 S 形曲线，关于 (0, 0.5) 中心对称，导数为 $\sigma'(z) = \sigma(z)(1 - \sigma(z))$。
- **交叉熵损失 (Cross-Entropy Loss)**：$L = -\frac{1}{n}\sum_{i=1}^{n}[y_i \log \hat{y}_i + (1-y_i) \log (1-\hat{y}_i)]$，来自最大似然估计，是分类问题的标准损失函数。
- **决策边界 (Decision Boundary)**：$\mathbf{w}^T\mathbf{x} + b = 0$ 对应的超平面，将特征空间划分为正类和负类区域。逻辑回归的决策边界是线性的。
- **最大似然估计 (MLE)**：选择参数使观测数据出现的概率最大化。在逻辑回归中，最大化似然等价于最小化交叉熵损失。
- **Softmax 函数**：$P(y=k|\mathbf{x}) = \frac{e^{\mathbf{w}_k^T\mathbf{x}}}{\sum_{j=1}^{K} e^{\mathbf{w}_j^T\mathbf{x}}}$，将多分类的输出向量归一化为概率分布。
- **对数几率 (Log-Odds)**：$\log\frac{p}{1-p} = \mathbf{w}^T\mathbf{x} + b$，逻辑回归中线性组合的含义是正类对负类的对数几率。
- **正则化**：逻辑回归同样支持 L1/L2 正则化，在损失函数中加入 $\lambda \sum |w_j|$ 或 $\lambda \sum w_j^2$，防止过拟合并控制模型复杂度。

## 知识正文
逻辑回归是分类任务的基石算法。其核心思想是将特征的线性组合通过 Sigmoid 函数压缩到 (0,1) 区间，输出解释为正类的概率。对于二分类问题，模型为 $P(y=1|\mathbf{x}) = \sigma(\mathbf{w}^T\mathbf{x} + b) = \frac{1}{1 + e^{-(\mathbf{w}^T\mathbf{x} + b)}}$。当概率大于 0.5 时预测为正类，否则为负类。

为什么逻辑回归不使用 MSE 作为损失函数？从优化角度看，将 Sigmoid 输出代入 MSE 会导致损失函数非凸，梯度下降可能陷入局部最优。从统计角度看，逻辑回归假设标签服从伯努利分布，最大似然估计自然导出交叉熵损失：对单个样本，似然函数为 $\hat{y}^y (1-\hat{y})^{1-y}$，取负对数得 $-y\log\hat{y} - (1-y)\log(1-\hat{y})$。

交叉熵损失的梯度简洁优美：$\frac{\partial L}{\partial w_j} = \frac{1}{n}\sum_{i=1}^{n}(\hat{y}_i - y_i)x_{ij}$。这与线性回归的梯度形式完全一致，区别仅在于 $\hat{y}_i$ 的定义不同。这一优雅性质使得逻辑回归的梯度下降实现与线性回归几乎相同，只需将预测函数从恒等映射替换为 Sigmoid。

决策边界是理解逻辑回归分类行为的关键。由 $\mathbf{w}^T\mathbf{x} + b = 0$ 决定，该超平面两侧分别预测为正类和负类。权重向量 w 的方向确定了决策边界的法向量，偏置 b 决定其与原点距离。正是因为决策边界是线性的，逻辑回归无法直接处理像 XOR 这样的线性不可分问题——需要通过特征工程（如添加 $x_1x_2$ 的交互项）将数据映射到更高维空间。

从二分类推广到多分类有两种主流策略。一对一加投票（OvR / One-vs-Rest）：为每个类别训练一个二分类器，将该类作为正类、其余作为负类，对 K 类共训练 K 个分类器。直接多分类（Softmax 回归 / 多项式逻辑回归）：用一个 Softmax 层同时输出 K 个类别的概率，损失函数为多类交叉熵 $-\sum_k y_k \log \hat{y}_k$。Softmax 回归的参数数量为 K×d，而 OvR 也是 K×d，二者参数规模相同，但 Softmax 在一次前向传播中产出所有类别概率，训练更高效。

逻辑回归的优点在于模型简单、可解释性强、训练速度快。每个特征的权重 $w_j$ 可直接解释：$w_j$ 越大，特征 $x_j$ 对正类概率的贡献越大；具体可解释为 $x_j$ 每增加一个单位，对数几率增加 $w_j$。这一优点使其在金融风控、医疗诊断等需要可解释性的领域广受欢迎。

## 关键公式
- **Sigmoid 函数**：$$\sigma(z) = \frac{1}{1 + e^{-z}}$$，其中 $z = \mathbf{w}^T\mathbf{x} + b$ 为线性组合值。
- **交叉熵损失**：$$L = -\frac{1}{n}\sum_{i=1}^{n}\left[y_i \log \hat{y}_i + (1-y_i)\log(1-\hat{y}_i)\right]$$，其中 $\hat{y}_i = \sigma(\mathbf{w}^T\mathbf{x}_i + b)$。
- **梯度**：$$\frac{\partial L}{\partial w_j} = \frac{1}{n}\sum_{i=1}^{n}(\hat{y}_i - y_i)x_{ij}$$，$\frac{\partial L}{\partial b} = \frac{1}{n}\sum_{i=1}^{n}(\hat{y}_i - y_i)$。
- **Softmax 函数**：$$P(y=k|\mathbf{x}) = \frac{e^{z_k}}{\sum_{j=1}^{K} e^{z_j}}$$，$z_k = \mathbf{w}_k^T\mathbf{x} + b_k$ 为第 k 类的线性组合。
- **对数几率**：$$\log\left(\frac{P(y=1|\mathbf{x})}{P(y=0|\mathbf{x})}\right) = \mathbf{w}^T\mathbf{x} + b$$。

## 示例
**示例 1：考试通过预测**  
设学习时间 x（小时）预测是否通过考试。训练数据：(10, 0), (20, 0), (30, 1), (40, 0), (50, 1), (60, 1)。拟合逻辑回归得到 $\hat{y} = \sigma(0.12x - 4.5)$。当 $x = 50$ 时，$z = 1.5$，$\sigma(1.5) = 0.818$ > 0.5，预测为通过。决策边界位于 $x = 4.5/0.12 = 37.5$ 小时处——学习超过 37.5 小时则预测通过。

**示例 2：鸢尾花多分类（Softmax）**  
对 Iris 数据集的三类鸢尾花做 Softmax 回归。输入特征为花萼长度与花瓣长度。训练后得到三组权重 $W_{3 \times 2}$。对于一朵新花，计算三个类别的 $z_k$ 值，经 Softmax 得到概率分布 [0.05, 0.82, 0.13]，预测为第二类。

## 常见误区
1. **"逻辑回归只能做二分类"**：错误。通过 OvR 策略或 Softmax 回归可以方便地扩展到多分类问题。Scikit-learn 的 LogisticRegression 默认就使用 OvR 支持多分类。
2. **"Sigmoid 输出就是置信度"**：需谨慎。Sigmoid 输出的概率值未经校准可能不准确（如 0.9 不代表 90% 的实际概率）。在需要可信概率的场景中应进行概率校准（如 Platt Scaling）。
3. **"逻辑回归要求特征线性可分"**：不全面。决策边界是线性的，但可以通过特征工程构造非线性特征（如多项式特征、交互项）来处理非线性问题，本质上等同于在原空间学习非线性边界。
4. **"逻辑回归对异常值不敏感"**：错误。虽然 Sigmoid 将输出压缩到 (0,1)，但极大或极小的异常特征值仍会显著影响权重估计。数据预处理中的异常值检测仍然重要。
5. **"多分类用 Softmax 一定比 OvR 好"**：不一定。OvR 中各分类器相互独立，可以并行训练，且某个类别的分类器出问题不影响其他类别。Softmax 中所有类别共享参数空间，可能存在竞争关系。

## 基础练习
**题目 1（单选）**：在逻辑回归中使用 MSE 作为损失函数的主要问题是：
A. 计算量太大
B. 损失函数非凸，容易陷入局部最优
C. 梯度会消失
D. 无法计算梯度
**正确答案**：B
**解析**：将 Sigmoid 输出代入 MSE 得到非凸损失函数，存在多个局部极小点，梯度下降不能保证找到全局最优。而交叉熵损失是凸函数，能保证收敛到全局最优。

**题目 2（单选）**：Softmax 回归中，若对所有权重同时加一个常数向量，输出概率会：
A. 改变
B. 不变
C. 取决于常数大小
D. 对某些类别改变
**正确答案**：B
**解析**：Softmax 的分母和分子同时缩放，由于 $e^{z_k + c} = e^{z_k}e^c$，分子分母的 $e^c$ 因子会抵消，概率不变。这说明 Softmax 参数存在冗余自由度，实际中通常通过正则化或约定某类别权重为零来解决。

## 进阶练习
**题目**：推导逻辑回归交叉熵损失函数的梯度表达式 $\frac{\partial L}{\partial w_j} = \frac{1}{n}\sum_{i}(\hat{y}_i - y_i)x_{ij}$。并解释为什么这个梯度形式说明预测值 $\hat{y}_i$ 和真实值 $y_i$ 的差距直接驱动了参数更新。

**参考答案**：
设 $\hat{y}_i = \sigma(z_i)$，$z_i = \mathbf{w}^T\mathbf{x}_i + b$，单个样本的损失为 $\ell_i = -y_i \log \hat{y}_i - (1-y_i)\log(1-\hat{y}_i)$。

由链式法则：$\frac{\partial \ell_i}{\partial w_j} = \frac{\partial \ell_i}{\partial \hat{y}_i} \cdot \frac{\partial \hat{y}_i}{\partial z_i} \cdot \frac{\partial z_i}{\partial w_j}$。

第一项：$\frac{\partial \ell_i}{\partial \hat{y}_i} = -\frac{y_i}{\hat{y}_i} + \frac{1-y_i}{1-\hat{y}_i}$。
第二项：$\frac{\partial \hat{y}_i}{\partial z_i} = \sigma(z_i)(1-\sigma(z_i)) = \hat{y}_i(1-\hat{y}_i)$。
第三项：$\frac{\partial z_i}{\partial w_j} = x_{ij}$。

相乘并化简：$\frac{\partial \ell_i}{\partial w_j} = \left(-\frac{y_i}{\hat{y}_i} + \frac{1-y_i}{1-\hat{y}_i}\right) \cdot \hat{y}_i(1-\hat{y}_i) \cdot x_{ij} = (-y_i(1-\hat{y}_i) + (1-y_i)\hat{y}_i) x_{ij} = (\hat{y}_i - y_i)x_{ij}$。

对所有样本取均值即得最终梯度。

这个形式说明：当 $\hat{y}_i$ 等于 $y_i$ 时梯度为零，参数不更新；当 $\hat{y}_i$ 偏离 $y_i$ 时，梯度正比于偏离程度，偏离越大更新幅度越大——这正是"错误驱动学习"的直观体现。
